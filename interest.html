Parkson Disease Predictor

In order to thoroughly undertand the ML algorithms behind K-nearest neighbours, I adapted online tutorials from the MIT OCW and coded in Python the actual KNN algorithm. This project makes use of the algorithm and two datasets on Parkison patients from UCI. The model is trained with the first dataset and then evaluated with the predictions on whether a given set of patient data frmo the second data set indicates Parkison disease.
